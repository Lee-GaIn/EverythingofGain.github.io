Overview of Git and GitHub









Git is an enterprise established in 2008 which provides services for the cooperation of software development by using Git.
The major services of them are hosting others to user's projects and managing versions of software easily. By using Git which is a system to track changes of files, programmers can manage files, track bugs and collaborate without location and time limitation. In other words, they cooperate with various programmers efficiently.


Github is an innovative service for developers regarding collaboration is a significant factor in the IT area. Many software and applications are invented by the efforts of a number of people. It will possibly be a burden as well as taking a long time to build the whole part of services or software alone. However, the application may be enhanced than the last version and released soon if other programmers and even students contribute to the project. Although publicizing a program could be a dangerous journey in that it is hard to generate revenue, it will probably not only bring the opportunity to develop other programmers but also improve the user's programme better. It means that sharing an individual's programme will probably get benefits in the long-term.


These days, many enterprises tend to publish their software or program as an open-source on Github. For example, TensorFlow is an open-source for machine learning released by Google. Through the platform, many developers study machine learning and contribute their efforts which is helpful for the development of the programme.
